# 命名规范

## 常规命名规则

|  Name   | Description | Template	| Exampl |
|---|---|---|---|
| Camel Case |驼峰。首单词首字母小写。剩余单词首字母小写。| camelCase	| quentinHsu |
|Kebab Case	| 短横线。单词首字母均小写，并以短横线连接。 | kebab-case	| quentin-hsu |
| Pascal Case |	帕斯卡。单词首字母均大写，依次排列，不用任何字符连接。	| PascalCase | QuentinHsu |
| Snack Case | 下划线。单词首字母均小写，并以下划线连接。 | snake_case | quentin_hsu |
